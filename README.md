# AspNet.Module.Test

Packages that simplify unit and integration testing for ASP.NET applications.

## Installation

```sh
dotnet add package AspNet.Module.Test.Abstractions
dotnet add package AspNet.Module.Test.Unit
dotnet add package AspNet.Module.Test.Int
```

## Packages

- `AspNet.Module.Test.Abstractions` provides shared helpers and testing contracts
- `AspNet.Module.Test.Unit` provides helpers for unit tests
- `AspNet.Module.Test.Int` provides helpers for integration tests
